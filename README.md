# Teste Prático de Sistema de Gestão de Funcionários

Este repositório contém a resolução de um desafio técnico.

## Tecnologias Utilizadas

- **Java 21**
- **API de Streams**
- **BigDecimal**
- **LocalDate**

## Requisitos Implementados

O sistema executa as seguintes operações conforme solicitado no desafio:

1.  **Inserção**: Cadastro de funcionários com nome, data de nascimento, salário e função.
2.  **Remoção**: Exclusão dinâmica de registros específicos (ex: "João").
3.  **Formatação**: Exibição de datas no padrão brasileiro (`dd/MM/yyyy`) e valores monetários com separadores de milhar e decimal.
4.  **Aumento Salarial**: Atualização em lote de salários (aumento de 10%).
5.  **Agrupamento**: Organização de funcionários por função utilizando `Map`.
6.  **Filtros**: Identificação de aniversariantes em meses específicos (Outubro e Dezembro).
7.  **Cálculos de Idade**: Lógica para identificar o funcionário mais velho.
8.  **Ordenação**: Listagem de funcionários em ordem alfabética.
9.  **Totalização**: Cálculo da folha de pagamento total e proporção de salários mínimos por colaborador.
